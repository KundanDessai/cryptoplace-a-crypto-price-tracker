# Cryptoplace - Crypto Price Tracker (React + CoinGecko API)

A cryptocurrency price tracking web application that displays real-time market data for popular cryptocurrencies. The app fetches live price data using the CoinGecko API and allows users to monitor cryptocurrency market trends.

##  Features
- Real-time cryptocurrency price tracking
- Search and view multiple cryptocurrencies
- Live market data including price and market trends
- Responsive user interface
- API integration with CoinGecko
- Secure login functionality

##  Tech Stack
- React
- JavaScript (ES6+)
- CoinGecko API
- Tailwind CSS / CSS
- Vite

##  Project Structure
- src/
- components/
- pages/
- App.jsx
- main.jsx

##  Installation & Setup

1. Clone the repository
- git clone https://github.com/KundanDessai/cryptoplace-a-crypto-price-tracker.git
2. Navigate to project folder
- cd cryptoplace-a-crypto-price-tracker
3. Install dependencies
- npm install
4. Run development server
- npm run dev
